Features
========

**Frame Flow** brings expressive documentation tools directly into Blender:

**Rich Text Formatting**
------------------------
Write bold, italic, or underlined notes inside node frames.

**Emoji and Icon Support**
--------------------------
Add symbols for quick visual tagging and category grouping.

**Linked Text Editor**
----------------------
Open a linked text datablock for each frame node in one click.

**Universal Support**
---------------------
Works across Shader, Geometry, and Compositor editors.
