Quick Start
===========

This short guide helps you get started with **Frame Flow** in minutes.

1. Open the **Shader Editor** (or Geometry/Compositor).
2. Create a **Frame** node and select it.
3. Use the **Frame Flow Panel** in the sidebar to:
   - Add formatted text (bold, italic, underline)
   - Insert emojis or icons
   - Open a linked text editor

Your changes are stored with the `.blend` file.
