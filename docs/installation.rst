Installation
============

Follow these steps to install **Frame Flow** in Blender:

1. Download the `.zip` file from Gumroad or Blender Market.
2. In Blender, go to **Edit → Preferences → Add-ons → Install...**
3. Select the downloaded `FrameFlow.zip` and enable the add-on.

**Troubleshooting**
-------------------
- If the add-on doesn’t appear, restart Blender.
- Remove any older Frame Flow versions to avoid conflicts.
