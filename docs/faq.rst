FAQ
===

**Q:** Which Blender versions are supported?  
**A:** Blender 4.0 and above.

**Q:** Can I use Frame Flow with Geometry Nodes?  
**A:** Yes, full support is available in the official version.

**Q:** Does it work offline?  
**A:** 100%. Frame Flow requires no internet access.

**Q:** Is there a free version?  
**A:** A limited version is available on the Blender Extensions site.
