Usage Guide
============

Installation
------------
1. Download the latest `FrameFlow` release from Gumroad or the Blender Extensions website.
2. In Blender, open **Edit → Preferences → Add-ons**.
3. Click **Install...**, then select the downloaded `.zip` file.
4. Enable the add-on named **Frame Flow**.

Features
--------
- Rich text formatting in frame titles (bold, italic, underline).
- Unicode icon and emoji support.
- One-click multiline text editor integration.
- Custom emoji database and quick search.

Quick Start
-----------
1. Create a **Frame** in the Node Editor.
2. Use the **Frame Flow Panel** to add formatted text or emojis.
3. Click the **Text Editor** icon to open the linked documentation block.
